WQV wristcam tool

The "WQV wristcam tool" project has the following goals:

  • Create a cross platform alternative to WQVLink so people can access the wristcam images from any platform(linux/os X)
  • Create tools to download and upload images to a wqv-2 casio wristcam.
  • Document the protocol used so that other implementations are possible.
  • Create tools to manage and publish the downloaded images.
