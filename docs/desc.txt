The WQV casio wristcam tool is a tool to download en edit images created with the casio wristcam wqv-2 watch . The application was written because at this point there are no good tools to download and administrer images created with the wrist cam. at this stage the software cam download the images from the watch and convert them to the xpm file format. Goal of the project is to enhance the gui to administer the downloaded images and publish them to the web. The software is written in java and will probabely work on any platform for wich there is a javax.comm package available or gnu RXTXcomm
